## 1. backend

- Init docker compose

```sh
# init postgres
docker-compose up
# init spring boot
./mvnw spring-boot:run
```

## 2. frontend

```sh
    # enter the ui project 
    cd book-network-ui
    # install the packages
    npm i
    # init server
    ng serve
```
